A comprehensive aim training application designed to help users improve their accuracy, reaction time, and precision in fast-paced targeting scenarios. Perfect for gamers, esports enthusiasts, and anyone looking to sharpen their hand-eye coordination.

## ğŸš€ Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:
- **Node.js** (v14.0.0 or higher)
- **npm** (v6.0.0 or higher) or **yarn**
- **Git** (for cloning the repository)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/hrishikhapekar/Aim-Trainer.git
   cd Aim-Trainer
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```
   Or if you prefer yarn:
   ```bash
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```
   Or with yarn:
   ```bash
   yarn start
   ```

4. **Open in browser**
   The application will typically open automatically at `http://localhost:3000/`. If not, manually navigate to this address in your web browser.

## ğŸ“– Usage

### Basic Gameplay

1. Launch the application
2. Select your preferred game mode from the main menu
3. Adjust difficulty and other settings as desired
4. Click "Start Game" to begin training
5. Follow on-screen instructions to complete the training session
6. Review your performance metrics at the end

### Game Modes

- **Classic Aim Mode**: Target and click appearing targets as quickly as possible
- **Reaction Time Challenge**: Test your reflexes with rapid-fire targets
- **Precision Mode**: Focus on accuracy over speed
- **Endurance Mode**: Extended training sessions for stamina building

### Performance Metrics

Track your progress with detailed statistics:
- **Accuracy %**: Percentage of successful hits
- **Reaction Time**: Average time to click targets
- **Targets Hit**: Total number of successful hits
- **Session Duration**: Length of your training session
- **Score**: Overall performance score
